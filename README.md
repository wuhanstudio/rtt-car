# 奔跑的服务器

这是一辆可以奔跑的网站服务器，下面图片上的网站本身就是搭建在小车上的。

![](./demo.png)

除了可以控制搭载 RT-Thread 本身的小车，也可以在网页上控制其他的小车。

![]()

这就是 2 辆小车的全照。

![](./robot.jpg)

另外，小车上还保存了通用机器学习 onnx 模型，可以在线进行手写体识别，根据识别的结果做出控制。

![](./onnx.png)
